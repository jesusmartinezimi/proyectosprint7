# proyectosprint7
¿Para qué sirve la aplicación web?

La aplicación web es un panel de control interactivo para analizar datos de anuncios de venta de coches. Es como crear un dashboard donde los usuarios pueden explorar y visualizar información sobre vehículos de manera dinámica.

Funcionalidades principales que proporcionará:

📊 Análisis Exploratorio de Datos:
- Visualizaciones interactivas con gráficos de Plotly

🌐 Interfaz Web Intuitiva:
- Aplicación web desarrollada con Streamlit
- Interfaz fácil de usar sin necesidad de programación para el usuario final
- Actualizaciones en tiempo real según las selecciones del usuario

☁️ Despliegue en la Nube:
- Aplicación accesible desde cualquier navegador
- Hospedada en Render.com para que otros puedan usarla

Objetivo del proyecto:

Este proyecto te permite practicar habilidades de ingeniería de software que son muy valoradas por los empleadores:

Gestión de entornos virtuales
Desarrollo de aplicaciones web
Control de versiones con Git/GitHub
Despliegue en servicios en la nube
